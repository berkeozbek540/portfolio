🚀 Personal Portfolio & Showcase

A modern, high-performance, and responsive portfolio website designed to showcase my technical skills, projects, and professional journey. Built with a focus on clean code, aesthetic design, and seamless user experience.

✨ Features
- Modern UI/UX: A sleek dark-themed interface built with Tailwind CSS and Shadcn/UI.
- Dynamic Interactions: Smooth animations and page transitions powered by Framer Motion.
- Multilingual Support: Ready for international audiences with integrated language switching (EN/TR).
- SEO Optimized: Structured with Next.js for high search engine visibility and performance.
- Responsive Design: Fully optimized for mobile, tablet, and desktop screens.

🛠️ Tech Stack
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Components: Shadcn/UI
- Animations: Framer Motion
- Deployment: Vercel
